Privacy



We don't send any of your images to external servers. All the image transcoding is done on your device. You can see the source code from our GitHub page if you want to be sure.

Hosting is provided by GitHub Pages. To make this page decent-looking, a font (Work Sans) is downloaded from Google Fonts' server. No data is shared with them.

image-converter saves on your device only the settings, and you can delete them at any time from your browser's settings.
